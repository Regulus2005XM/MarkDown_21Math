# the自定义宏

 **\def\hello{\overline{A}}** 

 **\hello 自定义**

显示：$\def\hello{\overline{A}} \hello$  

 **\def\greet#1{a\in \overline{#1}}** 

 **\greet{B} 含一个String类型参数的自定义**

显示：$\def\greet#1{a\in \overline{#1}}   \greet{B}$  

 **\def\s#1#2{\Sigma^{#1}_{#2}}** 

 **\s{a}{b} 含多个参数的自定义**

显示：$\def\s#1#2{\Sigma^{#1}_{#2}}   \s{a}{b}$
