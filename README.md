# MarkDown_21Math
MarkDown的一些基本语法与使用心得，并根据部分数学课程设置了速查表，方便记忆与使用

n个#+空格+headline 代表n级标题(1~6)

在内容两侧添加双\\*\*星号来加粗

在内容两侧添加单\*星号来加斜

用\\\>来块引用，在块中\\\-来加点，这两个都可以层层嵌套

用\\\[图片名字](/xxx/xxx.png)来插入图片

用\\\**\*号来创建分割线

用\[名](http)创造超链接

用\\\<br />来换行
